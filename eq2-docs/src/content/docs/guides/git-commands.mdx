## Git Command Basics
| Command | Description |
| ------- | ----------- |
| `git clone <repo>` | Clones a git repository to your local workspace |
| `git pull` | Pulls all changes online to Git |
| `git checkout -b feature/x` | Switches yourself into a branch |
| `git add .` | Adds all your files to a pending git commit |
| `git commit -m "message"` | Saves a local commit with a message |
| `git push` | Pushes your local commit to Git |
| `git status` | Diplays the current state of your working directory |
| `git log` | Displays commit history |
